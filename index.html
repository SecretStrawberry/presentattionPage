My Portofolio

All my projects are linked here, from the first to the last one.


While building my first website, I learned Ruby and Ruby on Rails. The website is a personal blog with cooking recipes, where users can not only read the blog, but sign in and add comments to the recipes.
The Admin can add new recipes and posts, plus have control over the users and comments. Since it was my starting project the user interface has a rough edge that I didn’t want to smooth out, so it shows where it all began. I’m very proud of it.
I know that form the point of view of User Interface is not that interactive, but because it is the first site that I have ever build, I am Proud of it and that is the reason I have add it in my portofolio.


The second website I built is for a small business located in the city I live. It is developed on Rails framework, and has the same characteristics as the first one, with specific modifications based on the user needs..
